Tutorial 5: Video Call 1 to N with WebRTC

This web application consists on an one to many video call using WebRTC. The Media Pipeline is composed by N interconnected WebRtcEndpoints. The first user entering a 'room' will become the 'master', i.e., the rest of the participants will be connected to this 'master' user. On the other hand, 'master' will be connected to the last user entering the 'room'. To run this demo follow these steps:

  1. Open this page with a browser compliant with WebRTC (Chrome, Firefox) and click on Start button.
  2. Grant the access to the camera and microphone. After the SDP negotiation the communication should start. When there is only one user connected, this communication is in loopback (mirror).
  3. In a different machine (or a different tab in the same browser) follow the same procedure. Every new user will see the first one, while the first one will see the last connected user.
  4. Click on Stop to finish the communication.
